# 刘子超个人网站 — 设计规范

## 设计方向

**关键词：** 暗色地图美学 · 文学质感 · 旅行者的笔记本

网站的视觉语言来自两个意象：**古旧地图**和**手写笔记**。暗色底色如同展开一张夜间的世界地图，墨色文字在纸面上浮现。不追求"现代感"或"科技感"，而是追求**纸墨的温度**和**旅途的沉静**。

整站支持暗色/明亮双主题。暗色模式为默认。明亮模式使用暖色羊皮纸色调。暖色强调色（accent 赤陶色）用于关键交互节点，像地图上用红笔标注的路径。

## 技术架构

- **单文件 HTML 架构**：index.html（主页）+ 4 个书详情页，所有 CSS/JS/SVG 内联
- **无框架、无构建工具**：纯 HTML/CSS/JS，可直接用任何 HTTP 服务器部署
- **主题系统**：`[data-theme="light"]` CSS 选择器覆盖所有 CSS 变量
- **i18n 系统**：`data-i18n` / `data-i18n-html` 属性 + JS `I18N` 对象
- **偏好持久化**：`localStorage` 键 `lzc-theme`（dark/light）和 `lzc-lang`（zh/en）
- **子页面同步**：4 个书详情页读取 localStorage 偏好，共享主题/语言状态

## 字体系统

| 用途 | 字体 | 权重 | 场景 |
|------|------|------|------|
| 英文展示 | Playfair Display | 400, 700, 400 italic | 标题、英文名、大标题 |
| 英文辅助 | EB Garamond | 400, 600, 400 italic | 标签、元数据、按钮文字、小号英文 |
| 中文正文 | Noto Serif SC | 300, 400, 600, 700 | 所有中文内容 |

**规则：**
- 中英混排时，中文用 Noto Serif SC，英文自动匹配 Playfair 或 EB Garamond
- 标签类文字统一用 EB Garamond，letter-spacing 0.1em+，uppercase
- 不使用 sans-serif。整站无衬线字体

## 页面结构

### index.html（主页，~1100 行）
1. **Hero 地图区**：SVG 世界地图 + 四段旅程路线动画 + 城市节点弹窗
2. **移动端旅程卡片**：替代 SVG 地图，scroll-snap 水平滚动
3. **关于区**：作者简介 + 抽象肖像 SVG 占位图
4. **作品区**：4 本书卡片（程序化 Canvas 封面 + 旅程色左边框）
5. **引用区**：3 条媒体评价
6. **译作区**：3 本译作卡片
7. **年表区**：13 个时间线条目（1984-2025）
8. **页脚**：社交图标（微博/豆瓣/邮箱）

### book-0N-*.html（书详情页，每页 ~260 行）
1. **Hero**：书封占位图 + 标题/描述/奖项
2. **路线**：SVG 城市连线图 + 城市标签
3. **影像**：Canvas 程序化占位图画廊
4. **书摘**：2-3 段原文摘录
5. **评价**：媒体/作家评语
6. **出版信息**：网格布局的元数据
7. **导航**：上/下一本书 + 返回地图

## 组件模式

### 区块（Section）
```
.sec — padding: 6rem 3rem (大屏) / 4rem 1.5rem (≤900px)
.sec-label — EB Garamond, 0.65rem, 大写, accent色, 带左侧横线
.sec-title — Playfair Display, clamp(1.6rem, 3.5vw, 2.4rem), 700
```

### 滚动揭示（Reveal）
```
.rv — opacity:0, translateY(30px), 0.7s cubic-bezier(0.16,1,0.3,1)
.rv.vis — opacity:1, translateY(0)
.rv1/.rv2/.rv3 — 延迟 0.1s/0.2s/0.3s
```

### 导航（Nav）
```
nav — fixed, flex, space-between, gap:2rem
.nav-logo — Playfair Display, 1rem, 700, nowrap, flex-shrink:0
.nav-links — flex, gap:2rem, EB Garamond uppercase
.nav-controls — flex, margin-left:auto（主题/语言切换按钮）
.nav-burger — 仅移动端显示（≤900px）
```

### 主题切换
```
#themeToggle — 太阳/月亮 SVG 图标
  暗色模式显示太阳图标，点击切换到明亮
  明亮模式显示月亮图标，点击切换到暗色
  偏好存储: localStorage.setItem('lzc-theme', 'light'|'dark')
```

### 语言切换
```
#langToggle — 显示 "EN" 或 "中"
  中文模式显示 "EN"，点击切换到英文
  英文模式显示 "中"，点击切换到中文
  偏好存储: localStorage.setItem('lzc-lang', 'zh'|'en')
  翻译数据: I18N 对象（~80 键/语言）
```

### 程序化占位图（makePlaceholder）
```
Canvas API 生成确定性图案：
  - 基于城市名种子的伪随机数
  - 渐变背景 + 等高线 + 网格点 + 曲线路径 + 暗角
  - 自动适配明暗主题（检测 data-theme 属性）
```

## 交互原则

- hover 状态使用 `transition: all 0.3s`
- 弹窗使用 `opacity` 过渡，0.4s
- 路线画线使用 `stroke-dashoffset` 动画，3s
- 不使用弹跳（bounce）或夸张的缓动
- `prefers-reduced-motion: reduce` 时禁用所有动画

## 可访问性

- 功能性文字最低对比度: WCAG AA (4.5:1)，使用 `--ink40` 及以上
- 触摸目标最小尺寸: 44×44px
- 所有交互元素可键盘聚焦
- SVG 地图城市节点: `role="button"` + `tabindex="0"` + `aria-label`
- 城市弹窗: 焦点陷阱 + Escape 关闭 + 焦点恢复
- 汉堡菜单: `aria-expanded` 状态切换
- 提供 `prefers-reduced-motion` 降级
