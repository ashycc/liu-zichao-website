# 刘子超个人网站 — 设计规范

## 设计方向

**关键词：** 暗色地图美学 · 文学质感 · 旅行者的笔记本

网站的视觉语言来自两个意象：**古旧地图**和**手写笔记**。暗色底色如同展开一张夜间的世界地图，墨色文字在纸面上浮现。不追求"现代感"或"科技感"，而是追求**纸墨的温度**和**旅途的沉静**。

整站使用全暗色调。暖色只出现在关键的交互节点上（accent 赤陶色），像地图上用红笔标注的路径。

## 色彩系统

### 基础色
| Token | 值 | 用途 |
|-------|-----|------|
| `--dark` | `#0f0e0b` | 主背景 |
| `--dark2` | `#1a1812` | 区块交替背景 |
| `--dark3` | `#252318` | hover / 卡片底色 |

### 墨色（文字）
| Token | 值 | 用途 |
|-------|-----|------|
| `--ink` | `#f5f0e8` | 标题、强调文字 |
| `--ink80` | `rgba(245,240,232,.8)` | 正文 |
| `--ink60` | `rgba(245,240,232,.6)` | 描述文字 |
| `--ink40` | `rgba(245,240,232,.4)` | 辅助文字 |
| `--ink20` | `rgba(245,240,232,.2)` | 微文字（图例、标签）|
| `--ink10` | `rgba(245,240,232,.1)` | 边框、分隔线 |
| `--ink05` | `rgba(245,240,232,.05)` | 极淡分隔 |

> **注意：** `--ink20` 对比度约 1.7:1，不满足 WCAG AA（4.5:1）。仅可用于纯装饰性文字，不可承载关键信息。承载功能信息时至少用 `--ink40`。

### 强调色
| Token | 值 | 用途 |
|-------|-----|------|
| `--accent` | `#c45a3c` | 交互元素、标签、线条 |
| `--accent-glow` | `rgba(196,90,60,.3)` | accent 光晕 |

### 旅程色
| Token | 值 | 对应旅程 |

## 字体系统

| 用途 | 字体 | 权重 | 场景 |
|------|------|------|------|
| 英文展示 | Playfair Display | 400, 700, 400 italic | 标题、英文名、大标题 |
| 英文辅助 | EB Garamond | 400, 600, 400 italic | 标签、元数据、按钮文字、小号英文 |
| 中文正文 | Noto Serif SC | 300, 400, 600, 700 | 所有中文内容 |

**规则：**
- 中英混排时，中文用 Noto Serif SC，英文自动匹配 Playfair 或 EB Garamond
- 标签类文字统一用 EB Garamond，letter-spacing 0.1em+，uppercase
- 不使用 sans-serif。整站无衬线字体

## 间距

使用以下标准间距值（基于 1rem = 16px）：

| Token | 值 | 典型用途 |
|-------|-----|---------|
| xs | 0.4rem | 元素内部间距 |
| sm | 0.8rem | 标签与标题间 |
| md | 1.5rem | 段落间距 |
| lg | 2.5rem | 区块标题到内容 |
| xl | 4rem | 区块间距（小屏） |
| 2xl | 6rem | 区块 padding（大屏） |

## 组件模式

### 区块（Section）
```
.sec — padding: 6rem 3rem (大屏) / 4rem 1.5rem (≤900px)
.sec-label — EB Garamond, 0.65rem, 大写, accent色, 带左侧横线
.sec-title — Playfair Display, clamp(1.6rem, 3.5vw, 2.4rem), 700
```

### 滚动揭示（Reveal）
```
.rv — opacity:0, translateY(30px), 0.7s cubic-bezier(0.16,1,0.3,1)
.rv.vis — opacity:1, translateY(0)
.rv1/.rv2/.rv3 — 延迟 0.1s/0.2s/0.3s
```

需要添加 `prefers-reduced-motion: reduce` 时禁用动画。

### 信息卡片（Map Info Card）
```
.map-info-card — 半透明暗底 + backdrop-filter:blur(8px)
  进入: opacity 0→1, translateY(20px→0), 0.5s
```

## 交互原则

- hover 状态使用 `transition: all 0.3s`
- 弹窗使用 `opacity` 过渡，0.4s
- 路线画线使用 `stroke-dashoffset` 动画，3s
- 不使用弹跳（bounce）或夸张的缓动

## 图片规格（待真实素材到位时使用）

| 类型 | 比例 | 建议尺寸 | 格式 |
|------|------|---------|------|
| 作者肖像 | 3:4 | 600×800px | JPG/WebP |
| 城市照片 | 3:2 | 900×600px | JPG/WebP |
| 书封 | 2:3 | 400×600px | JPG/WebP/PNG |

## 可访问性基线

- 功能性文字最低对比度: WCAG AA (4.5:1)
- 触摸目标最小尺寸: 44×44px
- 所有交互元素需可键盘聚焦
- SVG 交互区域需 ARIA 标记
- 弹窗需焦点陷阱 + 关闭后焦点恢复
- 提供 `prefers-reduced-motion` 降级
